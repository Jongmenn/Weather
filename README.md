# Weather

기상청 기상자료개방포털 원시자료

종관기상관측(Automated surface observing System, ASOS)

자료형태: 일 자료 

기간: 2000~2020

지역: 17개 시도 (관측지점)

기후평년값: 기후통계분석 > 평년값 > 우리나라 기후평년값 다운로드 

열대야/폭염일수: 기상일수

*열대야일수는 밤최저기온이 25 ℃ 이상인 날*
*기온이 밤에도 25 ℃ 이하로 내려가지 않을 때에는 너무 더워서 사람이 잠들기 어렵기 때문에 더위를 나타내는 지표로 열대야를 사용*


*폭염일수는 일 최고기온이 33℃ 이상인 날의 수를 의미 (기상청 기상현상일수)*
